Info

MathLamp Operators can be used for evaluating basic Arithmetic math.

Usage

Addition (+)

code: 1 + 1 -> 2
math: \(1 + 1\)

Subtraction (-)

code: 3 - 2 -> 1
math: \(3 - 2\)

Multiplication (*)

code: 2 * 2 -> 4
math: \(2 \times 2\)

Division (/)

code: 6 / 2 -> 3
math: \(6 \div 2\)

Modulus or Remainder (%)

code: 5 % 2 -> 1
math: \(5 \bmod 2\)
